Sobre o Projeto

Este blog foi projetado para avaliação de obtenção de nota na disciplina de Desenvolvimento de Softwares para Rede de computadores e Web. O "Código Simples" estará hospedado no respositório de um dos densenvolvedores e sempre ficará disponivel aqui. A source desse projeto foi idealizada como simples, porém de forma que agregasse todo conteudo pedido e permitisse uma navegação fluida por parte dos usuários assim como a retirada de processamento desnecessária.
